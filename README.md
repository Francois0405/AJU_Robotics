# 🤖 Club de Robótica - Asociación Joves Units Poble-sec (AJU)

¡Bienvenidos al repositorio oficial del Club de Robótica de **Joves Units del Poble-Sec**! 

Somos un punto de encuentro para mentes inquietas. Nuestra misión es democratizar el acceso a la tecnología y fomentar el aprendizaje colaborativo (*peer-to-peer*). 
Aquí no solo tiramos código; resolvemos problemas, trabajamos en equipo y creamos proyectos que van desde videojuegos retro hasta sistemas de automatización.

> **Nuestra filosofía:** "Aprender creando juntos!".

---

## 🚀 Proyectos Destacados

Cada carpeta de este repositorio contiene el código fuente, esquemas de conexión y documentación técnica de nuestras creaciones. Aquí tienes un resumen de lo que hemos hecho:

| Proyecto | Descripción en una línea |
| :--- | :--- |
| **🚀 Space Runner** | Un arcade de supervivencia espacial con música dinámica, sistema de disparo y guardado de récords en EEPROM. |
| **🧠 Quiz Master** | Un juego de trivia interactivo de 15 preguntas con navegación por joystick y optimización de memoria Flash. |
| *(Próximamente)* | *Nuevos proyectos de robótica móvil y sensores IoT.* |

---

## 🛠️ Stack Tecnológico Común

En el club utilizamos principalmente las siguientes herramientas:
- **Microcontroladores:** Arduino Uno, Nano, ESP32.
- **Lenguajes:** C++ (Arduino Framework), Markdown para documentación.
- **Componentes:** Pantallas LCD I2C, Joysticks analógicos, Sensores ultrasónicos, Servomotores y Buzzers.

## 🤝 Contribuir y Unirse

¿Eres del barrio y quieres aprender? ¿Eres un programador y quieres echarnos una mano con el código? 
1. **Haz un Fork** de este repositorio.
2. Crea una **nueva rama** con tu mejora (`git checkout -b feature/mejora`).
3. Envía un **Pull Request**.

O mejor aún, ¡pásate por la asociación y conócenos en persona!
