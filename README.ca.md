# 🤖 Club de Robòtica - Associació Joves Units Poble-sec (AJU)

Benvinguts al repositori oficial del Club de Robòtica de **Joves Units del Poble-Sec**!

La nostra missió és oferir accés a la tecnologia i fomentar l'aprenentatge col·laboratiu (*peer-to-peer*). Resolem problemes, treballem en equip i creem projectes que desperten les nostres ments i interès.

> **La nostra filosofia:** *"Aprendre creant junts!"*

---

## 🚀 Projectes Destacats

Cada carpeta d'aquest repositori conté el codi font i recursos detallats.

| Projecte | Descripció |
| :--- | :--- |
| **🤖 Claudek** | El robot principal del club. Compta amb múltiples sensors, capaç de seguir línies i ser controlat a distància. |
| **🕹️ Space Runner** | Un arcade de supervivència espacial amb música, sistema de tret i desament de rècords. |
| **🧠 Trivia** | Joc de trivia interactiu de 15 preguntes amb navegació mitjançant joystick. |

---

## 🛠️ Stack Tecnològic

Al club explorem el món *Maker* utilitzant principalment:

* **Microcontroladors:** Arduino Uno, Raspberry Pi 5, ESP32.
* **Llenguatges:** C++ (Arduino IDE) i Markdown per a documentació.
* **Components:** Pantalles OLED/LCD, joysticks, motors DC/Servos, sensors (ultrasons, IR), leds i electrònica bàsica.
